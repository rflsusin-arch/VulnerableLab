# Spring Vulnerable Lab

Laboratório educacional para discussão de segurança em aplicações Spring Boot.
Desenvolvido para a disciplina de Frameworks - Unipar.

## Executar

```bash
./mvnw spring-boot:run
```

Acesse em: http://localhost:8081  
Console H2: http://localhost:8081/h2-console

## ⚠️ Aviso
Este projeto contém vulnerabilidades **intencionais** para fins educacionais. Não use em produção.
